# Task Manager

Простое веб-приложение для управления задачами.

## Функции
- Создание задач с названием, описанием, категорией и приоритетом
- Отметка о выполнении
- Редактирование и удаление задач
- Поиск и фильтрация по категориям и приоритетам
- Статистика по задачам
- Тёмная тема
- Сохранение в localStorage

## Технологии
- HTML/CSS/JavaScript
- LocalStorage для хранения данных
- Чистый фронтенд (без бэкенда)

## Установка
1. Скачайте файлы
2. Откройте index.html в браузере
